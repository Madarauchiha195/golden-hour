# Golden Hour

**This is an Agentic AI, AI, and Machine Learning (ML) project.**

Golden Hour is a medical emergency coordination system that leverages Agentic AI to automate triage, risk prediction, traffic evaluation, hospital routing, ambulance dispatch, and case summaries. The system orchestrates multiple specialized sub-agents via the `lyzr-architect` framework.

## Features
- **Agentic AI Workflow**: A manager agent orchestrates a 10-step emergency pipeline with multiple specialized sub-agents.
- **Machine Learning Models**: Predicts survival probability and assigns risk scores.
- **Real-Time Dashboards**: Incident intake and workflow visualization with beautiful UI.
